# Skill: Revisión de Carpeta / Batch Processing Universal

## Contexto
Una ingestora automatizada de PDFs para abstraer resúmenes que compilarán introducciones y marcos teorícos independientemente del proyecto base, obedeciendo las directrices estrictas globales que limitan las iteraciones autocomplacientes.

## Instrucciones de Ejecución
1. Pidiendo una dirección centralizada, usa tus herramientas nativas de directorio para listar documentos analizables.
2. Genera las pasadas iterativas `view_file` para cada uno e invoca internamente a **Review Paper** con la flexibilidad de amoldarse al *style format* que decida la ruta maestra de formato (autor-año o referencias numeradas).
3. Escribe invariablemente cada informe en carpetas categorizadas. Termina con un chequeo total.
