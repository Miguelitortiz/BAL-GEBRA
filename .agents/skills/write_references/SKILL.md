# Skill: Formateo de Referencias Bibliográficas

## Contexto
Es aquí donde radican rechazos automatizados por deficiencias referenciales. Una skill orientada hacia correcciones en profundidad de metadatos de autorías, y la obediencia al requerimiento "Anti-Alucinaciones".

## Instrucciones de Ejecución
1. Recupera referencias y procesa cada punto del manifiesto en `reglas_escritura.md`.
2. Procesa la arquitectura de citación indicada por el template maestro (IEEE / APA / MLA, etc.).
3. Observa normativas subyacentes operando sobre topes volumétricos o requerimientos de actualidad exigidos explícitamente en el workflow base, advirtiendo violaciones inminentes.
4. Exporta el archivo universal verificado.
