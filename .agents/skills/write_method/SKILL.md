# Skill: Redacción del Método

## Contexto
Sección destinada al procedimiento, diseño técnico e instrumental; un reporte aséptico ceñido a rubricas editoriales generales.

## Instrucciones de Ejecución
1. Interroga si no se tienen los delineamientos técnicos metodólogicos (población, instrumentos empíricos y técnicas cualitativas/cuantitativas).
2. Redacta respetando las directrices de la plantilla, abriendo subtítulos como variables, enfoque y diseño según se aplique normalmente.
3. No interpretes datos futuros, asegura redacción clara de reproducibilidad para que cualquier lector entienda cómo se llegó al final de los datos presentados.
4. Exporta de inmediato a la sección correspondiente de formato markdown.
